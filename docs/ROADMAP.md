# 后续迭代路线

## V3.1 稳定性冻结

- Keil 全量构建和目标板回归。
- 栈高水位、Heap 余量、DMA/I²C 错误计数监控。
- 核实背光 PWM 与复位黑屏。
- UI 明确标识真实数据和模拟数据。

## V3.2 真实运动与电量

- 接入 IMU 和计步算法。
- 运动页使用真实步数、距离、热量和心率。
- 接入 ADC 或电量计，完成电量滤波和百分比曲线。

## V3.3 无线与天气

- 重构 ESP8266 AT 状态机。
- 接入 HTTP/MQTT、天气 JSON 和网络校时。
- 定义来电与消息通知协议。

## V3.4 资源与低功耗

- 压缩全屏图片和字体。
- 清理未使用的旧 MAX30102、AP3216C 和 cJSON 构建项。
- 屏幕、传感器、无线和 MCU Stop 模式协同。
- 记录待机、亮屏、测量、联网四类功耗。

## 发布条件

只有同时满足以下条件，才建议从 Alpha 升级为 Beta：

1. Keil 0 Error，主要 Warning 已解释。
2. 连续运行 24 小时无任务卡死。
3. 触摸、显示、I²C 压力测试通过。
4. 模拟数据在 UI 和文档中有明确标识。
5. 至少完成真实计步、电量或联网天气中的两项。
