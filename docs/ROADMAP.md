# 开发规划

## 1. 系统稳定性

- 完成冷启动、复位与长时间运行测试；
- 优化页面刷新与滑动体验；
- 检查任务栈水位与系统内存；
- 完善背光 PWM 控制。

## 2. 运动与电量

- 接入 IMU；
- 实现真实计步与姿态识别；
- 接入电池 ADC 或电量计；
- 完善距离、热量与电量模型。

## 3. 无线通信

- 接入 ESP8266；
- 建立 Wi-Fi 通信；
- 获取在线天气；
- 实现网络校时；
- 扩展消息与来电通知。

## 4. 健康功能

- 优化心率滤波与峰值检测；
- 调整 MAX30102 LED 电流与阈值；
- 完善信号质量判断；
- 继续验证血氧算法参数。

## 5. 低功耗

- 屏幕休眠与触摸唤醒；
- 传感器电源管理；
- 无线模块休眠；
- STM32 低功耗模式；
- 不同工作状态功耗统计。
