# 功能展示

## Home 页面

- 数字时间
- 日期与星期
- 指针表盘
- 温度与湿度
- 步数
- 电量
- 天气概览

## Health 页面

- 心率显示
- 血氧显示
- 心率波形
- 最大值与最小值
- 测量控制

## Sport 页面

- 运动模式
- 运动时间
- 距离
- 热量
- 运动心率

## Weather 页面

- 地点与日期
- 当前温度
- 天气状态
- 最高与最低温度
- 天气图标

## Call 页面

- 联系人信息
- 来电状态
- 接听与拒绝交互界面

## 系统功能

- FreeRTOS 多任务调度
- LVGL 页面与动画
- 左右滑动循环切换
- RTC 时间与表盘同步
- SPI1 + GPDMA 屏幕刷新
- FT6336 触摸输入
- SHT20 温湿度采集
- MAX30102 健康采集链路
- I²C 多设备互斥访问
- USART1 启动日志
